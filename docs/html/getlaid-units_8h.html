enum GTLD_UnitType

Possible unit types for GTLD_Unit.

Enumerator
GTLD_UNIT_PX 

Raw pixel size.

GTLD_UNIT_PERCENT 

Size relative to a container, where "1.0" is 100% of the container height/width.

GTLD_UNIT_AUTO 

Special unit type, all of the GTLD_UNIT_AUTO sized units fill the remaining space in the container evenly.
